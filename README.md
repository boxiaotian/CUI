# CUI

#### 介绍
小程序原生，自建UI库，专注于大型组件

##### 目录介绍

```
> components
  Item / 条目
  Navi / 左上CUI logo
  Title / 组件标题
  DatePicker / 公历农历组件
  
> pages
  index / 首页
  calender / 日历 日期类型页
  date / 日历表以及时间选择 （页面型组件）
  gl_nl / 公历农历选择
```
